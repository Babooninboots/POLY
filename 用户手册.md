# POLY — 交互式 LAMMPS 多晶样品生成器

作者：张杨 (Yang Zhang)

单位：Stony Brook University (纽约州立大学石溪分校)

## 概述

POLY 是一个基于 PySide6 的 GUI 应用程序，用于生成 LAMMPS 分子动力学模拟的多晶原子结构。
它生成完全周期性的 3D 多晶体，支持用户定义的晶粒尺寸分布、晶体结构和晶体学取向。

**后端流水线**（在后台线程中运行）：

1. **种子生成** — 通过随机、正态、双峰（高斯混合）、自定义、层状或均匀分布放置晶粒位置，用 FBSP 力偏球堆积和 pyvoro2 进行 Voronoi 胞元剖分。
2. **完美晶体** — 从 Bravais 晶格、金属间化合物、空间群数据、CIF 文件或 POLY `.crystal` 文件生成超胞。
3. **取向分配** — 通过随机、低/高角度、Z 轴对准、CSL、自定义取向差或自定义取向文件模式分配 Euler 角（ZXZ 约定）。
4. **多晶组装** — 最佳周期性镜像选择、逐晶粒旋转/平移、KD-tree Voronoi 裁剪、模包裹。
5. **LAMMPS 输出** — `.data`（原子数据文件）和 `.dump` 文件。

**前端**（GUI）：

- Voronoi 视图（PyVista 3D），支持交互式切片和点选。
- 完美晶体视图，CPK 着色原子。
- 晶粒尺寸和取向差直方图。
- 晶粒编辑器，可手动微调种子位置/取向。
- 后台工作线程，带进度条、暂停和停止。

## 分步工作流程

### 1. 设置模拟盒子

在 **盒子尺寸** 中，设置 X、Y、Z 的 `最小（起始）` 和 `最大（结束）` 坐标，单位为 Å。
默认：`(0,0,0)` → `(100,100,100)`。

> 对于层状和均匀分布配合 Z 轴对准模式，Z 尺寸会自动调整为对准方向上
> 晶体重复周期的整数倍。

### 2. 选择晶粒数量

在 **晶粒数量**（直接计数，1–10⁸）和 **平均晶粒直径**（Å）之间切换。
更改一个会根据盒子体积和等效球直径模型自动更新另一个。

### 3. 选择晶粒尺寸分布

| 分布类型          | 描述                       | 额外参数                       |
| ------------- | ------------------------ | -------------------------- |
| **随机**        | 均匀随机种子位置                 | —                          |
| **正态分布**      | 力偏球堆积，目标为对数正态尺寸分布        | 标准差 (Å)                    |
| **双峰**        | 高斯混合模型：两个正态分布按数量分数混合    | 模式 1 分数、两个均值、两个标准差；数量自动计算 |
| **自定义**       | 从 `.seed` 文件加载预定义种子位置    | 文件选择器                      |
| **层状**        | 中平面上的 2D 柱状晶粒，面内 Voronoi | 面内分布 (`random` 或 `normal`) |
| **均匀排列 (1D)** | 种子沿一个轴等间距排列，覆盖整个垂直平面     | 轴选择 (`x`, `y`, 或 `z`)      |

> 选择层状或均匀排列会自动将取向模式切换为 Z 轴对准。

### 4. 配置晶体结构

| 来源          | 字段                                                                                                 |
| ----------- | -------------------------------------------------------------------------------------------------- |
| **Bravais** | 结构 (`sc`, `fcc`, `bcc`, `hcp`, `diamond`, `tetragonal`, `bct`)，元素，a (Å)，c (Å)                      |
| **金属间化合物**  | 原型 (`L1₂`, `B2`, `D0₃`, `L2₁`, `D0₁₉`, `A15`, `C15`, `L1₀`, rocksalt, zincblende 等)，元素，a (Å)，c (Å) |
| **空间群**     | 元素，基元（分数坐标），空间群编号，晶胞参数 (a,b,c,α,β,γ)                                                               |
| **自定义（文件）** | `.cif`、`.crystal` 或 ASE 可读文件                                                                       |

#### 多相支持

点击绿色 **ADD CRYSTAL** 按钮创建额外的晶体相（Phase 1、Phase 2…）。
使用 **Phase** 选择框在相之间切换。每个相独立存储其晶体来源、结构、元素和晶格参数。

对于 > 0 的相，分配使用该晶体的晶粒：
- **Fraction**：占总晶粒数的小数比例（如 `0.3` = 30%）
- **Grain List**：逗号分隔的晶粒 ID，支持公式：
  - `2n` — 所有偶数索引晶粒 (0, 2, 4, …)
  - `2n+1` — 所有奇数索引晶粒 (1, 3, 5, …)
  - `d < 20`、`d > 50`、`d <= X`、`d >= X` — 按直径筛选
  - `1,3,5-8` — 显式 ID 和范围
  - 可组合：`2n, 5, d<30`

Phase 0（基体）自动获得所有剩余未分配晶粒。原子类型 ID 在相之间连续
（Phase 0 类型从 1 开始，Phase 1 类型从 Phase 0 的最大类型 ID 继续，以此类推）。

多相模式下保存完美晶体会写入 `{name}.0.crystal`、`{name}.1.crystal` 等文件。

六方结构的 `a` → `c` 比例会自动填充 (c/a ≈ 1.633)。

### 5. 选择取向模式

| 模式           | 描述                         | 额外参数                                                                           |
| ------------ | -------------------------- | ------------------------------------------------------------------------------ |
| **随机**       | 均匀随机 Euler 角               | —                                                                              |
| **Z 轴对准**    | 将晶体方向 `[hkl]` 对准模拟 Z 轴     | (hkl) 指数，面内模式 (`random`, `low_angle`, `high_angle`, `symmetric_csl`)，Σ 值，CSL 角 |
| **低角度**      | 从基础取向生成小角度随机取向差            | —                                                                              |
| **高角度**      | 大角度随机取向差                   | —                                                                              |
| **自定义取向差分布** | 从文件加载取向差角分布                | 文件选择器                                                                          |
| **自定义取向文件**  | 从 `.euler` 文件加载逐晶粒 Euler 角 | 文件选择器                                                                          |

**对称 CSL 模式**：当 `面内` 设置为 `symmetric_csl` 时，应用程序会计算给定 Σ 和
`[hkl]` 对应的所有 CSL 取向差角。从下拉列表中选择一个，将取向差对称分配
（例如，晶粒 0 获得 +θ/2，晶粒 1 获得 −θ/2，绕 Z 轴旋转）。
盒子尺寸会自动调整为每个垂直方向上晶体投影重复周期的精确整数倍。

### 6. 生成种子

点击 **生成初始种子**。后台线程会：

- 按所选分布放置种子。
- 运行 Voronoi 剖分（正态/层状正态分布用 Monte Carlo 优化）。
- 分配取向。
- 生成完美晶体超胞。

Voronoi 3D 视图显示按 Euler 角着色的多面体，图表更新晶粒尺寸和取向差分布。

### 7. 检查和编辑

- **点击晶粒** 在 Voronoi 视图中选中（高亮 + 线框立方体）。左侧面板的
  **选中晶粒编辑器** 会填充其位置、Euler 角和直径。
- **修改** 数值并点击 **应用编辑** 以就地重新计算 Voronoi 剖分和取向差，
  无需重新生成种子。
- **切片视图** 拖动切片深度滑块或循环切换视图按钮（等距 → X → Y → Z）。
- **调整透视** 使用透视滑块。

### 8. 保存中间状态（可选）

- **保存种子状态** 写入 `{Name}.seed`（N×3 种子位置，若使用了 FBSP 则为 N×4，
  含 Laguerre-Voronoi 半径）和 `{Name}.euler`（N×4：grain_id φ₁ Φ φ₂）。
- **保存完美晶体** 写入 `{Name}.crystal`（XYZ 格式，头部包含晶胞矢量）。

### 9. 配置输出

设置 **输出文件夹**（浏览…）和 **状态名称**。所有输出文件写入
`{Folder}/{Name}.ext`。

### 10. 构建多晶

点击 **开始构建**。后台线程：

1. 以所需覆盖度生成完整完美晶体超胞。
2. 运行多晶组装流水线（最佳周期性镜像、逐晶粒 KD-tree 裁剪、模包裹）。
3. 写入 `{Name}.data`（LAMMPS 原子数据）和 `{Name}.dump`（逐原子 grain_id + Euler 角）。

状态栏显示最终的原子和晶粒数量。

## 柱状结构（层状 / 均匀排列）

对于层状（2D 柱状）和均匀排列（1D）分布，组装流水线使用专门的柱状算法：

1. 完美晶体生成为边长 `2 × max_grain_diag` 的立方体。
2. 晶体预旋转，使 `[hkl]` 对准盒子 Z 轴（计算一次，所有晶粒共用）。
3. 对准后的晶体裁剪为圆柱柱体（直径 = `2 × max_grain_diag`，高度 = `1.5 × max_grain_z`）。
4. 逐晶粒：柱体按晶粒半径裁剪，面内旋转（Rz = R_total × R_base⁻¹），
   平移到种子位置（带 0.1 Å 余量偏移），KD-tree 裁剪。
5. 所有原子模包裹进入主模拟盒子。

## 文件格式

### `.seed` — 种子位置

三列格式（仅位置）：

```
# POLY seed state: {name}  |  {N} grains  |  distribution={dist}
# box_start: xlo ylo zlo
# box_end: xhi yhi zhi
x1 y1 z1
x2 y2 z2
...
```

四列格式（位置 + Laguerre-Voronoi 半径，使用 FBSP 优化时自动保存）：

```
# POLY seed state: {name}  |  {N} grains  |  distribution={dist}
# columns: x y z radius
# box_start: xlo ylo zlo
# box_end: xhi yhi zhi
x1 y1 z1 r1
x2 y2 z2 r2
...
```

使用 **自定义** 分布加载四列文件时，半径列会传递给 Laguerre-Voronoi
幂距离剖分，保留原始 FBSP 运行的精确胞元形状。

### `.euler` — Euler 角

```
# POLY orientation state: {name}  |  mode={mode}
# columns: grain_id phi1 Phi phi2 (zxz, degrees)
0 φ₁ Φ φ₂
1 φ₁ Φ φ₂
...
```

### `.crystal` — 完美晶体

```
# POLY crystal: {name}  |  {N} atoms  |  formula={formula}
# crystal_system: {cubic|hexagonal|tetragonal|orthorhombic|triclinic}
# cell_1: ax ay az
# cell_2: bx by bz
# cell_3: cx cy cz
# element  x  y  z
Mg  x1 y1 z1
...
```

`# crystal_system:` 头部指定晶体对称系统，用于正确的取向差计算。
从 GUI 保存时自动从结构类型或空间群检测。旧文件若缺失则默认为 `cubic`。

### `.data` — LAMMPS 原子数据

标准 LAMMPS 格式，含质量块。原子类型为 1 基索引，按元素排序。

### `.dump` — LAMMPS dump

逐原子字段：`id_POLY type x y z grain_id euler_angle_1 euler_angle_2 euler_angle_3`。
晶粒 ID 为 1 基索引。

## 提示

- 对于 **层状** 和 **均匀排列** 分布，始终使用
  **Z 轴对准** 取向。盒子 Z 尺寸将自动调整为 `[hkl]` 晶面间距的整数倍。
- **重投** 按钮清除所有缓存并强制完全重新生成。
  用它来尝试不同的随机种子（每次生成随机状态都会推进）。
- 种子优化期间的 **暂停** 按钮渲染当前 Monte Carlo 状态，方便检查收敛情况。
- 如果构建产生空晶粒，尝试增大盒子尺寸或减少晶粒数量。
  空晶粒意味着 Voronoi 胞元小于晶格间距。
- 要实现可复现的状态，保存 `.seed` 和 `.euler` 文件，然后用
  "自定义" + "自定义取向文件" 模式重新加载。

## 算法

### 种子分布优化

POLY 支持五种晶粒种子放置策略，其中最复杂的是用于正态分布和层状/正态
面内分布的 **力偏球堆积** (Force-Biased Sphere Packing, FBSP)。

#### 随机 / 自定义 / 均匀排列

种子直接放置，无需优化：

- **随机**：盒子内均匀采样。
- **自定义**：从 `.seed` 文件加载。
- **均匀排列 (1D)**：种子沿一个轴等间距放置，在垂直平面中居中。

#### 力偏球堆积 (FBSP)

在指定目标晶粒尺寸分布时使用（正态分布、双峰分布，或层状配合正态面内分布）。

1. **目标采样**：为每个晶粒分配目标直径。正态分布：从 N(target_mean, target_std)
   抽取。双峰分布：从高斯混合 φ·N(μ₁,σ₁) + (1−φ)·N(μ₂,σ₂) 抽取，其中 φ 为
   用户指定的模式 1 数量分数。截断为 ≥ 0.1 × mean（双峰为 0.1 × min(μ₁,μ₂)）。
2. **堆积分数缩放**：缩放直径，使总体积（3D，堆积分数 ≈ 0.58）或总面积
   （2D，堆积分数 ≈ 0.78）填满域，转换为等效 Laguerre-Voronoi 半径。
3. **迭代松弛**（最多 2000 步）：
   - KD-tree（在活动轴上带 PBC 包裹）查找 2 × max_radius 内的所有
     重叠球对。
   - 对每个重叠对 (i, j)，沿分离方向施加与 overlap × learning_rate
     成正比的排斥力，按 r_j/(r_i+r_j) 加权，使小球位移更多。
   - 学习率每步衰减：lr ← lr × 0.995。
   - 当 max_overlap < 0.01 Å 或无重叠时收敛。
4. **输出**：松弛后的种子位置 + 目标半径。对于双峰分布，最终晶粒直径
   直接取自目标半径（2 × r）以保留高斯混合信号；对于正态分布，直径
   从 Laguerre-Voronoi 胞元体积重新计算。黄色目标 PDF 曲线叠加在
   晶粒尺寸直方图上（正态为单高斯，双峰为高斯混合）。

算法支持暂停/恢复以实时检查收敛情况，以及提前停止供用户取消。

#### Voronoi 剖分

种子放置后，`pyvoro2` 计算 27 镜像周期性 Voronoi 剖分
（3×3×3 主盒子复本）。对于幂距离（Laguerre-Voronoi）剖分，来自 FBSP
的目标半径作为逐种子权重传递，确保每个 Voronoi 胞元紧密匹配目标晶粒体积。

### 取向差优化

#### BFS 图遍历（低角度 / 高角度）

用于完整 3D 低/高角度模式以及 Z 轴对准的面内扭转分量。

1. 随机选择一个起始晶粒，分配一个均匀随机的旋转。
2. 在 Voronoi 邻接图上进行广度优先搜索：
   - 对每个子晶粒，生成最多 200 个候选相对旋转
     （随机轴，角度取自目标范围）。
   - 每个候选与 **所有** 已分配邻居检查。
   - 低角度：所有逐对取向差必须 < 10°。
   - 高角度：所有逐对取向差必须 > 20°。
   - 接受第一个有效候选；若无候选通过，使用违规最小的候选。
3. 取向差计算为真实的 3D 逆向旋转角：
   `|R1 × R2⁻¹|`（取向差四元数的旋转角）。

#### 对称 CSL（重位点阵）

用于 Z 轴对准配合 `symmetric_csl` 面内模式：

1. 函数 `get_cubic_csl_angles(hkl, Σ)` 枚举所有 (m, n) 整数对，
   满足 m² + (h²+k²+l²)n² = Σ（去除因子 2 后）。CSL 取向差角为
   θ = 2 × arctan((n/m) × √N)。
2. 选定的角度对称分配：晶粒 0 获得 +θ/2，晶粒 1 获得 −θ/2 绕 Z 轴
   （双晶），或对多晶粒设置交替正负号。
3. 垂直方向的盒子尺寸自动调整为晶体投影重复周期的整数倍，确保
   CSL 边界完全周期性。

#### Monte Carlo 取向差匹配（自定义取向差）

用于匹配用户提供的目标取向差角分布：

1. **初始状态**：所有晶粒随机旋转。
2. **能量函数**：当前逐对取向差角分布与目标分布之间的
   Kolmogorov-Smirnov (KS) 统计量。小样本时回退到均值差。
3. **MC 循环**（最多 2000 步，模拟退火）：
   - 随机选取一个晶粒，施加小角度随机旋转扰动（σ = 5° 角度扰动）。
   - 计算新构型的 KS 能量。
   - 通过 Metropolis 准则接受/拒绝，温度 T 按 T ← max(0.001, T × 0.995) 衰减。
   - KS 统计量 < 0.05 时提前停止。
4. **输出**：找到的最佳旋转、能量历史和接受率。

### 晶粒组装（多晶构建）

组装流水线将逐晶粒取向和完美晶体超胞转化为最终的原子结构。

#### 标准流水线

1. **27 镜像 PBC 种子树**：每个晶粒的 N 个种子在 3×3×3 周期性偏移网格
   （每个方向 ±box）上复制，得到 27N 个种子位置的平坦数组。
   构建 cKDTree 用于最近种子查询。
2. **最佳周期性镜像**：对每个晶粒，选择距盒子中心最近的周期性镜像，
   最小化最终模包裹后的 Voronoi 胞元分裂。
3. **逐晶粒处理**：
   - **球形预裁剪**（在原点）：保留晶粒种子周围
     radius = 0.55 × grain_diagonal 内的完美晶格原子。
     （旋转保持距原点距离不变，因此在原点裁剪等价于旋转后裁剪。）
   - **旋转**：将完整旋转矩阵应用于裁剪后的原子。
   - **平移**：原子移动到最佳种子位置。
   - **KD-tree 查询**：每个原子对 27N 种子树查询。对于 Voronoi 裁剪，
     最近种子不是当前晶粒最佳镜像的原子被丢弃。对于 Laguerre-Voronoi，
     使用幂距离 (d² − r²) 和 k=30 最近种子。
4. **模包裹**：所有原子包裹进入主盒子：
   `pos = (pos − box_start) mod box_size + box_start`。

**晶粒级并行化**：当 ≥ 8 个晶粒且估计预裁剪 ≥ 500 原子/晶粒时，
通过 `ThreadPoolExecutor`（上限 8 线程）并发处理晶粒。每个晶粒的
裁剪、旋转、KD-tree 查询和掩码操作均释放 GIL，获得接近线性的加速。
结果按晶粒索引顺序收集，保证确定性输出。进度从主线程按完工顺序逐晶粒打印。

**自适应 KD-tree 并行化**：当估计预裁剪超过 5,000 原子/晶粒时，
`cKDTree.query` 调用使用 `workers=-1`（全部核心）内部并行化查询。
低于 5,000 时，`workers=1` 避免小查询集上的线程池开销。

#### 柱状流水线（层状 / 均匀排列）

当 `is_columnar=True` 且指定了 `[hkl]` 对准方向时使用：

1. **R_base 计算**：构建旋转矩阵，将晶体 `[hkl]` 方向对准模拟 Z 轴
   （堆叠轴）。通过扫描垂直于 z_crys 的低指数候选方向，构建正交晶体
   坐标系 (x_crys, y_crys, z_crys)。
2. **主柱体**：整个完美晶体通过 R_base 预旋转，然后裁剪为圆柱：
   半径 = max(grain_diagonals)，高度 = 1.5 × max_grain_z
   （任何 Voronoi 胞元的最大 Z 范围）。
3. **逐晶粒处理**：
   - **柱状预裁剪**：逐晶粒圆柱（半径 = grain_diagonal[g]，高度同主柱体）。
   - **面内旋转**：Rz = R_total × R_base⁻¹，在原点应用。
   - **平移**：移动到最佳种子位置 + 0.1 Å 余量。
   - **KD-tree 查询**：与标准流水线相同的最近种子过滤。
4. **模包裹**：与标准流水线相同。

柱状流水线利用柱状晶粒的对称性：晶体预对准一次（R_base 由所有晶粒共享），
仅面内旋转 (Rz) 逐晶粒变化。柱状预裁剪（基于半径）比球形预裁剪
对层状结构典型的高纵横比胞元更内存高效。

## 数据库

### 材料科学背景

POLY 使用标准的材料科学数据库和约定：

- **晶体结构**：Bravais 晶格（sc, fcc, bcc, hcp, diamond,
  tetragonal, bct）和金属间化合物原型（L1₂/Ni₃Al, B2/CsCl,
  D0₃/Fe₃Al, L2₁/Heusler, D0₁₉/Mg₃Cd, A15/Cr₃Si, C15/MgCu₂, L1₀/CuAu,
  rocksalt/NaCl, zincblende/ZnS, cesiumchloride/CsCl, fluorite/CaF₂,
  wurtzite/ZnS-hex）通过 `PristineCrystal` 模块内置。
- **CSL（重位点阵）**：通过 `get_cubic_csl_angles()` 函数为立方晶体
  枚举 CSL 边界，该函数求解 Diophantine 方程 m² + N·n² = Σ 的整数 (m, n) 对，
  其中 N = h² + k² + l²。这涵盖了立方晶界的标准 Read-Shockley CSL 理论。
- **Euler 角**：ZXZ 约定（φ₁, Φ, φ₂），单位为度，与织构分析和 EBSD
  （电子背散射衍射）数据中广泛使用的 Bunge 约定一致。
- **CPK 着色**：原子使用 Jmol 风格的 CPK（Corey-Pauling-Koltun）颜色
  渲染，便于元素识别。
- **原子质量**：内置周期表质量 (g/mol)，ASE（原子模拟环境）在可用时
  作为首选运行时来源。

from workers import PolycrystalBuildWorker

worker = PolycrystalBuildWorker(

)
worker.finished.connect(lambda result: print(f"完成：{result.n_atoms} 个原子"))
worker.error.connect(lambda msg: print(f"错误：{msg}"))

该 worker 自动：

1. 从多面体数据计算逐晶粒 Voronoi 对角线和 `max_grain_z`。
2. 确定安全的晶体覆盖度（`2 × max_grain_diag`）。
3. 根据分布类型选择标准或柱状组装流水线。
4. 写入 LAMMPS 输出文件。

        & (pos[:, 2] >= box_s[2]) & (pos[:, 2] <= box_e[2]))

print(f"盒子内原子数：{in_box.sum():,} / {len(pos):,}")
print(f"最小/平均/最大保留：{result.keep_counts.min():,} / "
      f"{result.keep_counts.mean():.0f} / {result.keep_counts.max():,}")

# 检查空晶粒
empty = result.keep_counts == 0
if empty.any():
    print(f"警告：{empty.sum()} 个空晶粒")
